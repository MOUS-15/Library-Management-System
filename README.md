# Library-Management-System
​[English Version] 🇬🇧
​📚 Library Management System (Python)
​A comprehensive, console-based Library Management System developed using Python. This project demonstrates core programming concepts including data structures (Dictionaries), functional programming, and robust user input validation.
​🚀 Key Features:
​Dynamic Book Management: Add, delete, and search for books by ID, Name, or Author.
​Borrowing Logic: Efficiently track borrowed vs. available books with real-time status updates.
​Smart Search: Advanced search functionality that handles duplicate names and provides detailed matches.
​Data Integrity: Robust error handling for user inputs to prevent system crashes.
​User-Friendly Interface: Clear, interactive console menus with intuitive navigation.
​🛠️ Tech Stack:
​Language: Python 3.x
​Data Structure: Nested Dictionaries for efficient data retrieval.

​📚 نظام متكامل لإدارة المكتبة (بايثون)
​نظام برمجي متكامل لإدارة الكتب والمستعيرين، تم بناؤه بلغة بايثون كأداة تعليمية وعملية قوية. يركز المشروع على تطبيق مفاهيم البرمجة المتقدمة وإدارة البيانات بكفاءة عالية.
​🚀 المميزات الرئيسية:
​إدارة شاملة للكتب: إضافة وحذف الكتب مع إمكانية البحث المتطور بـ (الكود، الاسم، أو المؤلف).
​نظام الاستعارة الذكي: تتبع دقيق لحالة الكتب (متاح / مستعار) وتحديث الحالة تلقائياً.
​معالجة البيانات الضخمة: القدرة على التعامل مع تكرار الأسماء وعرض نتائج تفصيلية للمستخدم.
​حماية النظام: تضمن دوال التحقق من المدخلات عدم توقف البرنامج عند إدخال بيانات خاطئة.
​واجهة تفاعلية: تصميم قوائم واضحة تسهل على المستخدم التنقل بين خدمات المكتبة.
​🛠️ التقنيات المستخدمة:
​اللغة: Python 3.x
​هيكلة البيانات: القواميس المتداخلة (Nested Dictionaries) لسرعة الوصول للمعلومات.
