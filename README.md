# 📚 Library Management System

### **[English Version]** 🇬🇧
A comprehensive, console-based Library Management System developed using **Python**.

**🚀 Key Features:**
* **Dynamic Book Management:** Add, delete, and search for books.
* **Smart Search:** Advanced functionality that handles duplicate names.
* **User-Friendly Interface:** Clear, interactive console menus.

---

### **[باللغة العربية]** 🇪🇬
# نظام متكامل لإدارة المكتبة (بايثون)
نظام برمجي متكامل لإدارة الكتب والمستعيرين، تم بناؤه بلغة **بايثون**.

**🚀 المميزات الرئيسية:**
* **إدارة شاملة:** إضافة وحذف الكتب والبحث المتطور.
* **نظام الاستعارة:** تتبع دقيق لحالة الكتب (متاح / مستعار).
* **واجهة تفاعلية:** قوائم واضحة تسهل التنقل.
